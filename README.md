u14nodall
================

Shippable CI image for node.js on ubuntu 14.04 with commonly used services. Available versions are:


  1.  4.8.7
  2.  5.12.0
  3.  6.11.5
  4.  7.10.1
  5.  8.9.4
  6.  9.11.1


Services:

Please refer [u14all](https://github.com/dry-dock/u14all) to know supported services.
